# 🔐 Secure Auth Dashboard

A modern authentication dashboard built using React and TypeScript. This project provides secure user authentication with login, signup, and protected routes.

---

## 🚀 Features

- User Signup and Login
- Form Validation
- Protected Routes
- Dashboard Interface

---

## 🛠️ Tech Stack

- React
- TypeScript
- CSS

---

## ▶️ Run Locally

npm install  
npm run dev
